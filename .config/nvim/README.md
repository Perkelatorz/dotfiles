# Purpleator Neovim Config

A fully-featured Neovim setup with custom Purpleator theme, LSP, AI completion, and modern tooling.

## Quick Start

- **Leader:** `<Space>`
- **Which-key:** Press `<leader>` and wait ~500ms, or `<leader>?` to open anytime
- **Toggle theme:** `<leader>ct` (Purpleator ↔ Nightfox)

## Highlights

- **Purpleator** – Custom purple undertone theme with semantic syntax colors
- **Mason** – LSP, formatters, linters (gopls, svelte-language-server, etc.)
- **Codeium** – Free AI completion (ghost text)
- **Checkpoint system** – `<leader>vc` / `vr` to checkpoint/restore (great for AI edits)
- **Flash** – Jump labels (`s`, `S`, `f`/`F`/`t`/`T` for go-to-char)
- **Telescope** – Fuzzy find (files, grep, buffers, diagnostics)
- **LazyGit** – Git TUI via `<leader>lg`
